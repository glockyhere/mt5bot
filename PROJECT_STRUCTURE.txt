MT5 Trading Bot - Project Structure
====================================

mt5bot/
│
├── Core Files
│   ├── trading_bot.py          # Main entry point - orchestrates all components
│   ├── mt5_connector.py        # MT5 API wrapper - handles all MT5 communications
│   ├── trade_executor.py       # Executes trades and manages positions
│   ├── risk_manager.py         # Risk management and position sizing
│   └── logger_config.py        # Logging configuration and utilities
│
├── Strategy Framework
│   ├── strategy_base.py        # Abstract base class for all strategies
│   └── strategies.py           # Built-in strategy implementations:
│                                 - Moving Average Crossover
│                                 - RSI Strategy
│                                 - Bollinger Bands
│                                 - MACD Strategy
│
├── Configuration
│   ├── config.yaml             # Main configuration file
│   ├── .env.example            # Environment variables template

├── Documentation

│   ├── QUICK_START.md          # 5-minute setup guide
│   └── PROJECT_STRUCTURE.txt   # This file
│

│       ├── trading_bot.log     # Main application log
│       └── trades.log          # Trade execution log
│
└── .gitignore                  # Git ignore rules


Component Relationships
=======================

trading_bot.py
    ├── Uses: mt5_connector.py (for MT5 communication)
    ├── Uses: risk_manager.py (for risk calculations)
    ├── Uses: trade_executor.py (for order execution)
    ├── Uses: strategies.py (for trading signals)
    └── Uses: logger_config.py (for logging)

trade_executor.py
    ├── Uses: mt5_connector.py
    └── Uses: risk_manager.py

strategies.py
    └── Inherits from: strategy_base.py


Execution Flow
==============

1. User starts: python trading_bot.py
2. trading_bot.py loads config.yaml and .env
3. Connects to MT5 via mt5_connector.py
4. Initializes risk_manager.py with risk parameters
5. Initializes strategy from strategies.py
6. Main loop:
   a. Fetches market data via mt5_connector.py
   b. Generates signal from strategy
   c. Validates risk via risk_manager.py
   d. Executes trade via trade_executor.py
   e. Manages open positions
   f. Logs activities via logger_config.py
7. On shutdown: closes connections and logs summary


Quick Command Reference
=======================

Start trading:
  python trading_bot.py

Check status:
  python trading_bot.py --status

Run backtest:
  python example_backtest.py

Custom config:
  python trading_bot.py --config my_config.yaml

Stop bot:
  Press Ctrl+C (graceful shutdown)
